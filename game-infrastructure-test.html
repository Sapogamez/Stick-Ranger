🎮 Stick Ranger - Game Infrastructure

TypeScript-based game engine with React integration

🎯 Features Implemented:

  • ✅ TypeScript configuration with strict mode
  • ✅ React 19+ support with type definitions
  • ✅ Canvas rendering engine
  • ✅ Game loop with FPS monitoring
  • ✅ Entity Component System architecture
  • ✅ Build pipeline with npm scripts
  • ✅ Debug controls and monitoring

🎮 Debug Controls:

  • Space - Pause/Resume game
  • 1 - Time scale 0.1x
  • 2 - Time scale 0.5x
  • 3 - Time scale 1.0x (normal)
  • 4 - Time scale 2.0x
  • 5 - Time scale 5.0x

📁 File Structure:

src/
├── index.ts              # Main entry point
├── engine/
│   ├── GameLoop.ts      # Game loop with timing
│   └── Canvas.ts        # Canvas rendering
├── components/
│   └── Game.tsx         # React game component
└── systems/
    └── EntitySystem.ts  # Entity management
